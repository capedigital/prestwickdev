Teach reading skills that will stick well beyond test day.

Increase students' standardized test scores by teaching them how to identify the most important information in everything they read.

Each book in the Preparation for State Reading Assessments series contains 35 nonfiction reading selections with multiple-choice and open-ended comprehension questions modeled after those used on standardized tests, including the FCAT 2.0, STAR/CAHSEE, ACT, and ITBS exams.

Every Teacher's Edition explains the intended purpose for assessment tests and provides answers to each exercise with succinct explanations.

As an added bonus, each book also provides reading tips and strategies that work-not only for taking standardized tests, but also for gleaning important information from any reading material.

Download a Sample

How does this program improve test scores and sharpen comprehension skills?

  • Test formats, passage type, text length, and questions are similar to those on the most widely used assessment instruments.
  • Questions explore author's purpose, main idea, tone, context clues, inferences, and more key skills.
  • The wide selection of passages provides nonfiction reading practice.
  • With test-question familiarity, students are more comfortable and able to accurately demonstrate their abilities and earn better test scores.
